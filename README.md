# 🇮🇹 五一意大利深度游攻略

> 罗马 Roma + 佛罗伦萨 Firenze ｜ 七天六夜 ｜ 交互式在线旅行攻略

一个纯前端、开箱即用的意大利旅行攻略网页。以每日行程时间线为主线，集成交互地图、景点详情、交通与美食信息，界面采用手绘杂志风设计，手机 / 电脑均可流畅浏览。

## ✨ 在线预览

开启 GitHub Pages 后访问：**https://heiner0619.github.io/wayboox/**

## 🗺️ 功能亮点

- **每日行程时间线**：5.1–5.6 逐日安排，含时间点、交通、景点、餐饮
- **交互式地图**：基于 Leaflet 本地库，标注每日景点位置与路线
- **景点详情卡**：历史背景、门票用时、拍照建议与避坑提示
- **手绘杂志风 UI**：自定义字体 + 涂鸦装饰，响应式适配移动端
- **纯静态、无需后端**：任何静态托管即可上线

## 🚀 本地运行

由于使用了本地地图资源，建议用本地服务器打开（直接双击 `index.html` 也可，但地图瓦片可能受限）。先进入项目目录，再启动服务：

```bash
# 1. 进入项目目录（改成你自己 clone 后的路径）
cd wayboox

# 2. 用 Python 3 起一个本地服务器
python3 -m http.server 8000
```

启动后在浏览器打开 **http://localhost:8000** 即可。

> 没装 Python 也可以用 Node：`npx serve`，或直接用 VS Code 的 Live Server 插件打开 `index.html`。
